# 提示词批量导入工具

## 功能说明
该工具用于将 JSON 格式的提示词备份文件批量导入到网文助手系统的数据库中。

## 使用方法

### 前提条件
1. 确保网文助手系统已经运行过，数据库已初始化
2. 确保 JSON 文件存在于以下位置之一：
   - `d:\Downloads\提示词整体备份_2026-4-7.json`
   - 项目根目录下的 `提示词整体备份_2026-4-7.json`

### 运行导入
在项目根目录执行：
```bash
node import-prompts.js
```

### 导入效果
导入成功后，提示词将出现在网文助手系统的"提示词管理"模块中，按分类组织：
- **写作要求** 分类：
  - 写作要求
  - 爆款知乎风，一次 1w，生成停止请使用续写继续写
  - 女频写作 - 情感共鸣
  - 爆款写作 -【番茄风】
  
- **写作风格** 分类：
  - 乐子文风格

## 注意事项
1. 如果数据库中已存在同名同分类的提示词，将会被更新
2. 导入完成后需要重启网文助手系统才能看到最新变化
3. JSON 文件格式必须与备份文件一致

## 文件说明
- `import-prompts.js` - 导入脚本
- `提示词整体备份_2026-4-7.json` - 提示词备份文件（需自行放置）
